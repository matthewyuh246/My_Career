# 職務経歴書

## 1. 株式会社スリーシェイク

| 項目 | 内容 |
| --- | --- |
| 期間 | 2025年5月～6月（2ヶ月）、2025年10月～現在（2026年6月） |
| 使用技術 | AWS, Azure, GCP, Kubernetes, Terraform, Go, TypeScript など |
| 会社URL | https://3-shake.com/ |
| ブログ記事URL | https://sreake.com/blog/dive-deep-into-kubernetes-admission-control |

**内容:**

5月～6月はSreake事業部（SRE）に所属し、KubernetesやTerraform、Grafana、AWSなどを用いたSRE関連の技術検証を実施。

10月から現在にかけてはSecurify事業部のクラウドリソースのセキュリティ診断を開発するSBOM/CSPMチームに所属し、AWSやGCP, Azure, OCI, Cloudflare, SakuraCloudを中心にクラウドとの連携設定、リソース収集、セキュリティポリシーに基づく診断ロジックの実装（Go, TypeScript）や、各サービス（ECRやACRなど）のSBOM生成、コードレビュー、診断項目の見直し、既存のバグ修正、データバッチ、パフォーマンスチューニング、AI Agentを使用した開発サイクルの構築などの開発業務に従事。また、社内ブログにてKubernetesに関する技術記事を執筆。
